# Customer Churn Analysis Dashboard

## Business Problem
A subscription-based business was experiencing rising customer churn but had no clear understanding of why customers were leaving. Management needed to identify at-risk segments and develop targeted retention strategies.

## Live Dashboard
🔗 **[View Interactive Dashboard on Tableau Public](https://drive.google.com/file/d/1ygYIYHqpfSRdR9BJgm9sQal0gPc1mvTT/view?usp=sharing).

## Key Findings

| Finding | Business Insight |
| :--- | :--- |
| **Month-to-month contracts** have 3x higher churn than annual contracts | Offer incentives for long-term commitments |
| **Customers with tenure < 6 months** are most likely to churn | Implement onboarding and early engagement campaigns |
| **Higher monthly charges correlate with higher churn** | Improve value communication and offer premium support |
| **Senior citizens churn at lower rates** | Existing retention strategies are working well |

## Dashboard Features

| Sheet | Purpose |
| :--- | :--- |
| Churn Rate Scorecard | Overall churn percentage |
| Churn by Contract Type | Compare month-to-month vs annual contracts |
| Churn by Tenure | Identify high-risk customer segments by age |
| Churn by Monthly Charges | Price sensitivity analysis |
| Customer Demographics | Breakdown by gender, senior status, partnership |

## Filters Available
- Contract Type
- Tenure Range
- Gender
- Senior Citizen Status

## Recommended Actions
1. **Loyalty Pricing** - Offer discounts for annual contract commitments
2. **Early Engagement** - Proactive outreach in first 6 months
3. **Premium Support** - Dedicated support for high-value customers
4. **Churn Prediction** - Implement ML model for early warning
